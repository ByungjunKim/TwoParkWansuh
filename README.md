# 두 박완서, 작가 연구의 제도와 젠더 - 2010년대의 두 <여성문학연구>를 중심으로

### 1. 설명
이 레포지토리에는 논문에 쓰인 연구 데이터, Python 코드(ipynb)를 공유하고, 지면에서 확인하기 어려웠던 데이터 분석 결과(표/그림)를 results 폴더에 정리하였다.  

논문 심사가 끝나고, 게재 확정이 되면 연구 데이터를 체계적으로 더 정리하고 설명을 적을 예정이다.
